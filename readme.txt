Eemie's Crops for Json Assets - Version 1.2.0
Adds some new crops to the game.

:: Required Mods::
SMAPI (https://smapi.io/)
Json Assets (https://www.nexusmods.com/stardewvalley/mods/1720)

:: What This Mod Adds::
-Six varieties of pumpkins
-Six varieties of melons

:: Installation Instructions::
1. Download and install the latest version of SMAPI
2. Download and install the latest version of Json Assets
3. Copy '[JA] Eemies Crops' folder to Mods
4. You're done! Run the game.
